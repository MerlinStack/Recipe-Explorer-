# 🍳 Recipe Explorer App

A modern, responsive recipe discovery web application built with React and Material UI.

## ✨ Features

- ✅ Fetch and display 50+ recipes from DummyJSON API
- ✅ Responsive grid layout (1/2/4 columns)
- ✅ Recipe cards with images, ratings, and details
- ✅ Color-coded difficulty badges
- ✅ Loading skeletons and error handling
- ✅ Pagination with 12 recipes per page
- ✅ Smooth animations and hover effects
